# HVILE Design Learnings

## 目的

北欧家具ペルソナ（30–45歳、デザイン志向）は、「静けさと品格」を開いた瞬間に感じたい。HVILEでは、ウォームリネン背景・Cormorant Garamondの品格・スプリットヒーローで、スカンジナビアの朝の光のような体験を設計した。

HVILEは架空ブランドであり、実在の店舗・商品ではない。

## 設計したこと

- ヒーローは全面写真ではなく **スプリット2カラム**（左：エディトリアルテキスト、右：写真）にした。Kinfolk誌的な静けさが全面写真より出せる。
- `THE ART / OF BEING / HOME.` で "HOME." だけをセリフイタリック+セージグリーンにし、主役を作った。
- マーキーを素材名（Oak, Linen, Ceramic…）のセリフイタリックにし、製品リストではなく素材哲学として見せた。
- フィロソフィーセクションは縦書きラベルで「読む軸」を作り、Kinfolk的な大余白を実現した。
- カーソルドットをウォームオーク（#B5703A）にすることで、テーマカラーへの参照を細部に埋め込んだ。

## CSS

主要な判断:

```css
:root {
  --bg:        #F7F3ED;   /* ウォームリネン。冷たい白を避けた */
  --bg-dark:   #1C1915;   /* 木炭。純黒より木材感がある */
  --accent:    #3A5537;   /* ノルウェーの森。北欧自然への参照 */
  --warm:      #B5703A;   /* オーク色。カーソルと詳細アクセント */
}

.hero {
  display: grid;
  grid-template-columns: 1fr 1fr;   /* スプリット */
}

.hero-h1 {
  font-family: var(--font-serif);
  font-weight: 300;   /* 細め。重さではなく品格で見せる */
  line-height: 0.9;   /* タイトな行間でエディトリアル感 */
}
```

ライトテーマで品格を出すには、色を増やすのではなく **素材感（grain, --bg-sub）と行間の使い方** が鍵。

## フォント

- Display: `Cormorant Garamond weight 300` — 極細セリフで贅沢感と静けさを両立
- Body: `Outfit` — ジオメトリック。Inter/DM Sansを意図的に避けた

セリフとサンズの組み合わせで「品格×現代性」を表現。セリフを大きく使うほど余白の重要性が上がる。

## アニメーション

- ローダーはダーク背景でブランド名のみ。家具の「重さ」をスピードで表現しない。
- ヒーロー文字は行ごとにスライドアップ（ストリートより遅め：1.1s）。
- 写真は scale 1.06→1 で9秒かけてセトル。見ている間に「落ち着く」感覚。
- スクロールリビールは0.9s。急がない。

## マテリアルセクション

2×2グリッドの縦横比を意図的にずらした（1番と4番が4/3、2番と3番が3/4）。整列しすぎず、でも崩れすぎない張力を作っている。

## apparelシリーズとの対比

| 軸 | ARCH（street） | HVILE（nordic furniture） |
|----|---------------|--------------------------|
| テーマ | ダーク | ライト |
| Display font | Clash Display（幾何学） | Cormorant Garamond（セリフ） |
| アクセント | サイバーライム #C8FF00 | ノルディックセージ #3A5537 |
| ヒーロー | 全面写真+巨大タイポ | スプリット（テキスト/写真） |
| アニメーション速度 | 速い・攻撃的 | 遅い・有機的 |
| grain opacity | 0.04 | 0.025 |
| ペルソナ | 19–24歳ストリート層 | 30–45歳デザイン志向層 |
