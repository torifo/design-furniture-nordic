# HVILE — design-furniture-nordic

> **架空ブランド注記**: HVILEは本デザイン研究のために作成した架空ブランドです。実在のブランド・店舗・商品ではありません。

---

## Overview

**furniture design research series** の「北欧」ペルソナ作品。

「Hvile」はノルウェー語で「安らぎ」を意味する。デザイン意識が高い住まい手（30〜45代）が求める「静けさと品格」を、Kinfolk誌的エディトリアル余白×スプリットヒーローで実現する。

## Design Approach

アパレルデザイン研究（ARCH/LUEUR等）と同じ骨格パターンを持ちながら、色・フォント・テンポがペルソナによって別物になることを検証した最初の家具作品。

- **スプリットヒーロー**: 50/50 grid — 全面写真よりKinfolk的な静けさ
- **Cormorant Garamond weight 300**: 細さで品格を表現。重さでなく繊細さ
- **ウォームリネン背景**: 北欧の自然光・木材感。冷たい白を避けた
- **カスタムカーソル**: ウォームオーク `#B5703A` のドット
- **grain overlay**: `opacity: 0.025` でウッドテクスチャ的質感
